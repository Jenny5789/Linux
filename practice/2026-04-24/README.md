## 📝 [편집의 마법 (o, A, r)]

🚩 목표: 입력 모드(i)로 가기 전 단축키로 시간 단축

💻 [실습 텍스트]

Monday
Tuesday
Wednesday

 ✅ 미션:

1. 'Monday' 줄에서 'o'를 눌러 그 아래에 'New Day'를 써봅니다.
2. 'Tuesday' 끝에서 'A'를 눌러 바로 글자를 추가해 봅니다.
3. 'Wednesday'의 첫 글자 'W'에 커서를 두고 'r'을 누른 뒤 'M'을 쳐서 'Mednesday'로 바꿔봅니다.

---

## 📝 [화면의 분할(:vs)]

🚩 목표: 두 파일을 동시에 보며 작업하기

 ✅ 미션:

1. 'vi file1.txt'를 엽니다.
2. 명령 모드에서 ':vs'를 입력합니다. (화면이 반으로 갈라집니다!)
3. 'Ctrl + w'를 두 번 눌러 왼쪽과 오른쪽 창을 왔다 갔다 해봅니다.

---

## 📝 [최종 실전!]

💻 [실습 텍스트]

ServerName: localhost
Port: 8080
Status: Off

 ✅ 미션:

1. Status를 'On'으로 'r' 명령어를 써서 바꿉니다.
2. Port: 8080 줄을 복사(yy)해서 바로 아래 붙여넣고(p), 숫자를 9090으로 바꿉니다.
3. '/Server'로 검색해서 해당 줄로 이동합니다.
4. 모든 작업을 마치고 ':wq'로 저장합니다.
